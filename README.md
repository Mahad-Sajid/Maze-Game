# Maze Game (C++)

A console-based Maze Game developed in C++ where players navigate through a randomly generated maze to reach the destination. The game uses dynamic memory allocation, path validation, and a recursive backtracking algorithm to determine the shortest possible path from start to finish.

## Features

* Random maze generation using 0s (blocked paths) and 1s (walkable paths)
* Interactive player movement using W, A, S, D controls
* Real-time maze visualization with player position tracking
* Detection of unreachable destinations
* Recursive backtracking algorithm for shortest path discovery
* Move counter and win condition tracking
* Dynamic memory management using pointers

## Technologies Used

* C++
* Dynamic Arrays
* Recursion & Backtracking
* Pathfinding Algorithms

This project demonstrates fundamental concepts of data structures, recursion, dynamic memory allocation, and game logic implementation in C++.
